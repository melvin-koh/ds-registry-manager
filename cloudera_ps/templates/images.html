Additional image details available from Docker Registry

The table above shows repository path, tag count, and latest tag using the catalog and tags APIs. Additional metadata can be retrieved per image/tag via the manifest and config blob endpoints:

  • Manifest digest — unique SHA256 content hash from the manifest HEAD request
  • Image size — total bytes from summing manifest layer sizes
  • Architecture / OS — platform info from the config blob (e.g. amd64, linux)
  • Created date — image build timestamp from the config blob
  • Media type — OCI or Docker schema format
  • Layer count — number of layers in the manifest
